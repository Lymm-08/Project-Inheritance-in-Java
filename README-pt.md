# Projeto Acadêmico - Herança em Java

Este repositório contém um projeto acadêmico desenvolvido em **Java** utilizando a **Eclipse IDE**, com o objetivo de demonstrar os conceitos de **Herança** e **Polimorfismo** aplicados em um sistema bancário simplificado.

## Estrutura do Projeto
- **model**
  - `Conta.java` — Classe base que define atributos e operações comuns de uma conta bancária.
  - `ContaCorrente.java` — Classe derivada que implementa regras específicas para contas correntes.
  - `ContaPoupanca.java` — Classe derivada que implementa regras específicas para contas poupança.
- **view**
  - `Principal.java` — Classe responsável pela execução do programa e interação com o usuário.

## Objetivos
- Aplicar os princípios de **herança** entre classes para reutilização e organização de código.
- Implementar operações bancárias básicas, como **depósito** e **saque**.
- Demonstrar o uso de **polimorfismo** em cenários práticos, permitindo que diferentes tipos de contas sejam manipulados de forma genérica.
